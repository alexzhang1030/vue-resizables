# vue-resizables

The resizable utilities of Vue@3.

## Installation

```bash
pnpm i vue-resizables
```

## Features

### Directives

```vue
<script setup lang="ts">
import { Resizable as vResizable } from 'vue-resizables'
</script>

<template>
  <div v-resizable></div>
</template>
```
